# NutriVision

AI-powered nutrition intelligence — analyze any food instantly via text, voice, label scan, or food image.

## Features
- 🥗 Text, voice, label scan & food image analysis
- 🤖 AI-powered nutrition breakdown (no API key required)
- 📊 Health score, macro tracking & exercise equivalents
- 🌙 Dark mode, responsive design

## How to Run Locally
Just open `NutriVision.html` in any modern browser — no setup needed.

## How to Deploy
See the `nutrivision-netlify` folder for Netlify deployment instructions.
